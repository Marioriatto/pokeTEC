# pokeTEC

Un juego de rol inspirado en Pokemon, donde la presicion de las acciones no está determinada y las probabilidades juegan en tu contra.

## Requisitos

Instalar la fuente Pocket Monk de ./assets/fonts

## Modulos a Instalar
```bash
pip install pillow
```
